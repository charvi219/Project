# Person REST API

This is a RESTful API built with Node.js and MongoDB for managing a collection of people.

## Features
- Add a new person
- Retrieve all people
- Update person details
- Delete a person by ID or name

## Technologies Used
- Node.js
- Express.js
- MongoDB
- Mongoose
